# Detailed Setup Guide for Event Manager

1. Install software: Visual Studio(2017), Microsoft SQL Server Management Studio 17
2. Download application
3. FInd event-manager.sql file and extract it to MSSM Studio
4. Run Visual studio and start the project

## Getting Started

Start the application and if you already have an account you can log.
If not you can register or use example accounts:
 username: Test12 or username: pesho12, both having password: Test12

## Functions

Read all events,
Log or register to have ability to:
create event,
delete event
or
update event

You can delete or update only events, created by you.
